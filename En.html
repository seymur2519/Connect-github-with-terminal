How to pull and use a file from Github via terminal?

Firstly we need to connect the terminal to GitHub:

  • Open the Terminal
  • Write config --user.name "seymur2519" and send the username.
  • Write config --user.email "seymur2519@gmail.com" and send the email.
  • Write config --user.password "********" and send the password.(ALL OF THESE ARE WRITED ONLY ONCE.)
DOWNLOAD FILES FROM GITHUB:
  • Write ls and see the files in the directory.
  • Write cd Documents and enter the Documents folder.
  • Write mkdir githubtest and create githubtest folder.
  • Write cd githubtest and enter the githubtest folder.
  • Write clone https://github.com/seymur2519/Connect-To-Github.git and drag the files from github to the githubtest file on the computer
    • Name of the pulled folder is FirstProject.
  • Write cd FirstProject and enter the FirstProject folder.
  • Write code . and open the file there with the VS code program.
UPLOAD FILES TO GITHUB:
    Come to the directory where we downloaded the files through the terminal.
  • Write git add . and add contents.
  • Write git status and look at the status of the directory.
  • Write git commit -m "Rəy" and send the comment.
  • Write git push origin main and send the files to main room.
    • BONUS:
  • Write git pull origin main and pull the changes to memory of computer.

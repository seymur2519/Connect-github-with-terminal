Githubdan terminal vasitəsilə necə fayl çəkib istifadə etmək olar?

İlk öncə terminalı GitHub-a bağlamalıyıq:

  • Terminalı açırıq
  • Terminalda config --user.name "seymur2519" yazıb istifadəçi adını göndəririk
  • Terminalda config --user.email "seymur2519@gmail.com" gmaili göndəririk.
  • Terminalda config --user.password "********" yazıb şifrəni göndəririk.(BUNLARIN HAMISI YALNIZ BİR DƏFƏ YAZILIR.)
GİTHUBDAN FAYLLARIN YÜKLƏNMƏSİ:
  • Terminalda ls yazıb kataloqdakı faylları görürük.
  • Terminalda cd Documents yazıb Documents faylına keçid edirik.
  • Terminalda mkdir githubtest yazıb githubtest qovluğunu yaradırıq.
  • Terminalda cd githubtest yazıb githubtest qovluğuna keçid edirik.
  • Terminalda clone https://github.com/seymur2519/Connect-To-Github.git yazıb githubdakı faylları kompyuterdəki githubtest faylına çəkirik
    • Çəkiıən qovluğun adı FirstProject-dir
  • Terminalda cd FirstProject FirstProject qovluğuna keçid edirik.
  • Terminalda code . yazıb VS code programıyla oradakı faylı açırıq.
FAYLLARIN GİTHUBA YÜKLƏNMƏSİ:
  • Terminal vasitəsilə faylları yüklədiyimiz kataloqa gəlirik.
  • Terminalda git add . yazıb içindəkiləri əlavə edirik.
  • Terminalda git status yazıb kataloqun vəziyyətinə baxırıq.
  • Terminalda git commit -m "Rəy" yazıb rəy göndəririk.
  • Terminalda git push origin main yazıb faylları main otağına göndəririk.
    • BONUS:
  • Terminalda git pull origin main yazıb dəyişiklikləri kompyuterin yaddaşına çəkirik.
